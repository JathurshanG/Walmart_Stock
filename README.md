# Walmart_Stock

  Walmart est une entreprise américaine transnationale spécialisé dans la grande distribution. Elle est connue comme étant le leader dans ce domaine. Il est donc normal qu'elle soit coté en bourse et qu'elle soit l'une des grande entreprise du NYSDE.
  
  L'analyse suivante permet de décrire, les actions de Walmart.
  
  Nous nous interessons ici a 4 variables :
  
  - au prix de l'action à l'ouverture des marchés ( Open)
  - au prix de l'action à la fermeture des marchés (Close)
  - au prix le plus élevé de l'action ( High)
  - au prix le plus bas de l'action ( Low)
  
  Le ratio HV permet de determiné le prix maximum d'une action. dans notre cas les premières valeurs pour ce ratio est donnée par le tableaux suivant  :
